## Project Overview ğŸ‰

E-Commerce project made with React for self learning. I used <a href="https://bradfrost.com/blog/post/atomic-web-design">
Atomic Design</a> for components architecture. I used <a href="https://www.datocms.com/">Dato CMS</a> for handling products data.

## Tech/framework used ğŸ”§

- React
- Gatsby
- React Hooks
- React Toastify
- GSAP
- Styled Components
- Formik
- datoCMS
- Local Storage
- Husky | lint-staged
- Eslint | prettier
- Atomic design

## Features

- Browsing products fetched from datoCMS
- Adding/removing products from cart
- Filtering and sorting products by names, sizes and prices
- Changing quantity of products with same ID and size
- Viewing full details of products
- Responsive Web Design
